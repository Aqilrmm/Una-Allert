Una Allert Demo

A lightweight, customizable alert and toast notification library

Introduction

Una Allert is a modern, lightweight JavaScript library that provides beautiful alert dialogs and toast notifications for your web applications. It's highly customizable, easy to use, and has zero dependencies.

This demo page shows all the features of Una Allert and includes code examples to help you get started quickly.

Installation

To use Una Allert, include the CSS and JavaScript files in your HTML: 

<link rel="stylesheet" href="Una Allert.css">
<script src="Una Allert.js"></script>

Then initialize the library after the DOM is loaded:

<script>
  document.addEventListener('DOMContentLoaded', function() {
    unaAlert.init();
  });
</script>

Basic Alerts

Una Allert provides different types of alerts: success, info, warning, error, and question.

Code Example:

// Show a success alert
unaAlert.show('success', 'Success!', 'Your action was completed successfully.');

// Show an info alert
unaAlert.show('info', 'Information', 'Here is some important information for you.');

// Show a warning alert
unaAlert.show('warning', 'Warning', 'Please be careful with this action.');

// Show an error alert
unaAlert.show('error', 'Error', 'Something went wrong. Please try again.');

Confirmation Dialogs

Confirmation dialogs ask the user to confirm an action and execute a callback function when confirmed.

Code Example:

            // Basic confirmation dialog
            unaAlert.confirm(
                'Confirm Action', 
                'Are you sure you want to proceed with this action?',
                function() {
                    // This function will be called when the user clicks Confirm
                    console.log('User confirmed the action');
                    unaAlert.toast('success', 'Confirmed', 'Your action has been confirmed');
                }
            );

            // Confirmation with custom button text
            unaAlert.confirm(
                'Delete Item', 
                'Are you sure you want to delete this item? This action cannot be undone.',
                function() {
                    console.log('User confirmed deletion');
                    unaAlert.toast('success', 'Deleted', 'Item has been deleted');
                },
                {
                    confirmText: 'Delete',
                    cancelText: 'Keep'
                }
            );

            // Confirmation with vertical buttons
            unaAlert.confirm(
                'Subscribe to Newsletter', 
                'Would you like to subscribe to our weekly newsletter?',
                function() {
                    console.log('User subscribed');
                    unaAlert.toast('success', 'Subscribed', 'Thank you for subscribing!');
                },
                {
                    confirmText: 'Subscribe',
                    cancelText: 'No, thanks',
                    verticalButtons: true
                }
            );

Toast Notifications

Toast notifications are small, non-blocking messages that appear at the bottom of the screen and automatically disappear after a few seconds.

Code Example:

// Show a success toast
unaAlert.toast('success', 'Success', 'Operation completed successfully');

// Show an info toast
unaAlert.toast('info', 'Information', 'Here is some useful information');

// Show a warning toast
unaAlert.toast('warning', 'Warning', 'Please be careful with this action');

// Show an error toast
unaAlert.toast('error', 'Error', 'Something went wrong');

// Show a toast with custom duration (8 seconds)
unaAlert.toast('info', 'Custom Duration', 'This toast will stay for 8 seconds', 8000);

Advanced Usage

Una Allert provides additional features for more advanced use cases:

Programmatically Close Alerts 

// Close the current alert dialog
unaAlert.close();

Close Toasts Programmatically 

// Create a toast and get a reference to it
const myToast = unaAlert.toast('info', 'Downloading', 'Your file is downloading...');

// Later, you can close it programmatically
myToast.close();
