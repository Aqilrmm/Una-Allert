Demo Una Allert

Library notifikasi alert dan toast yang ringan dan dapat disesuaikan

Pengenalan

Una Allert adalah library JavaScript modern dan ringan yang menyediakan dialog alert dan notifikasi toast yang indah untuk aplikasi web Anda. Library ini sangat dapat disesuaikan, mudah digunakan, dan tidak memiliki dependensi.

Halaman demo ini menunjukkan semua fitur Una Allert dan mencakup contoh kode untuk membantu Anda memulai dengan cepat.

Instalasi

Untuk menggunakan Una Allert, sertakan file CSS dan JavaScript dalam HTML Anda: 

<link rel="stylesheet" href="Una Allert.css">
<script src="Una Allert.js"></script>

Lalu inisialisasi library setelah DOM dimuat:

<script>
  document.addEventListener('DOMContentLoaded', function() {
    unaAlert.init();
  });
</script>

Alert Dasar

Una Allert menyediakan berbagai jenis alert: sukses, info, peringatan, galat, dan pertanyaan.

Contoh Kode:

// Tampilkan alert sukses
unaAlert.show('success', 'Sukses!', 'Tindakan Anda berhasil dilakukan.');

// Tampilkan alert info
unaAlert.show('info', 'Informasi', 'Berikut adalah informasi penting untuk Anda.');

// Tampilkan alert peringatan
unaAlert.show('warning', 'Peringatan', 'Harap berhati-hati dengan tindakan ini.');

// Tampilkan alert galat
unaAlert.show('error', 'Galat', 'Terjadi kesalahan. Silakan coba lagi.');

Dialog Konfirmasi

Dialog konfirmasi meminta pengguna untuk mengonfirmasi tindakan dan mengeksekusi fungsi callback saat dikonfirmasi.

Contoh Kode:

// Dialog konfirmasi dasar
unaAlert.confirm(
    'Konfirmasi Tindakan', 
    'Apakah Anda yakin ingin melanjutkan tindakan ini?',
    function() {
        console.log('Pengguna mengonfirmasi tindakan');
        unaAlert.toast('success', 'Dikonfirmasi', 'Tindakan Anda telah dikonfirmasi');
    }
);

// Konfirmasi dengan teks tombol kustom
unaAlert.confirm(
    'Hapus Item', 
    'Apakah Anda yakin ingin menghapus item ini? Tindakan ini tidak dapat dibatalkan.',
    function() {
        console.log('Pengguna mengonfirmasi penghapusan');
        unaAlert.toast('success', 'Dihapus', 'Item telah dihapus');
    },
    {
        confirmText: 'Hapus',
        cancelText: 'Simpan'
    }
);

// Konfirmasi dengan tombol vertikal
unaAlert.confirm(
    'Langganan Newsletter', 
    'Apakah Anda ingin berlangganan buletin mingguan kami?',
    function() {
        console.log('Pengguna berlangganan');
        unaAlert.toast('success', 'Berlangganan', 'Terima kasih telah berlangganan!');
    },
    {
        confirmText: 'Langganan',
        cancelText: 'Tidak, terima kasih',
        verticalButtons: true
    }
);

Notifikasi Toast

Notifikasi toast adalah pesan kecil yang tidak mengganggu, muncul di bagian bawah layar dan akan hilang secara otomatis setelah beberapa detik.

Contoh Kode:

// Tampilkan toast sukses
unaAlert.toast('success', 'Sukses', 'Operasi berhasil diselesaikan');

// Tampilkan toast info
unaAlert.toast('info', 'Informasi', 'Berikut informasi yang bermanfaat');

// Tampilkan toast peringatan
unaAlert.toast('warning', 'Peringatan', 'Harap berhati-hati dengan tindakan ini');

// Tampilkan toast galat
unaAlert.toast('error', 'Galat', 'Terjadi kesalahan');

// Tampilkan toast dengan durasi kustom (8 detik)
unaAlert.toast('info', 'Durasi Kustom', 'Toast ini akan bertahan selama 8 detik', 8000);

Penggunaan Lanjutan

Una Allert menyediakan fitur tambahan untuk kasus penggunaan yang lebih kompleks:

Menutup Alert Secara Programatik 

// Tutup dialog alert saat ini
unaAlert.close();

Tutup Toast Secara Programatik 

// Buat toast dan simpan referensinya
const myToast = unaAlert.toast('info', 'Mengunduh', 'File Anda sedang diunduh...');

// Kemudian, Anda bisa menutupnya secara manual
myToast.close();
